Conversion Engine Dark Mode Extracted Layers

Included:
- Cropped PNGs: each dashboard/photo/card visual extracted as its own asset.
- Full-size layer PNGs: same 1672x941 canvas with everything transparent except that extracted visual, useful if you want to layer it back over the original design.

Note: because the original design is dark mode and the cards themselves are dark, I preserved the internal dark surfaces instead of trying to auto-remove all dark pixels. That avoids destroying shadows, borders, and dashboard detail.
